Atom

Atom Text Editor

A lot of people are also starting to use Atom as their goto text editor.

Go ahead and download it. Open the .dmg file, drag-and-drop in the Applications folder, you know the drill now. Launch the application.

Atom is not free, and as a comprehensive list of packages and themes.

After installation you can open a file with $ atom myfile.py or start a new project in the current directory with $ atom .. Pretty cool. We'll configure Sublime more in the next few sections.
